A Pratt parser works by maintaining an **operator precedence table**—essentially a data structure mapping operators to their parsing rules:

  "+": { precedence: 1, parse: (left) => /* parsing logic */ },
  "*": { precedence: 2, parse: (left) => /* parsing logic */ },

function parseExpression(minPrecedence = 0) {

}
